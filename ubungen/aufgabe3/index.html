Thema Arrays: Zähle wahre Werte

			Schreiben Sie eine Funktion, die true (nicht truthy!) Werte aus den verschiedenen Arrays zählt.
			Prüfen Sie die Ergebnisse in der Console.
			
			
			
			Beispiel:
			console.log(checkTrue(arr1)) // 2

			console.log(checkTrue(arr2)) // 0

			console.log(checkTrue(arr3)) // 0

			etc.
