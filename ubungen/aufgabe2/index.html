Thema Objekte: Berechne Sekunden

 

			Schreiben Sie eine Funktion, die Stunden in Sekunden konvertiert.
			1. Lesen Sie das Eingabefeld aus.
			2. Geben Sie das Ergebnis im #info Element aus.
			3. Binden Sie die Funktion an den Button.
			
			
			Beispiel:
			"1 Stunde sind 3600 Sekunden"
			aber:
			"2 Stunden sind 7200 Sekunden"
