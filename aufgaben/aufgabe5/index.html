Variablen / Arrays

 

	Um Variablen zu verstehen, muss man sie genau betrachten.
	
	Schreiben Sie ein Object mit einer Methode,  die aus dem Array arr per Klick  
	alle Werte nacheinander ausliest und im p#info Element ausgibt.

	Sorgen Sie für folgende Reihenfolge pro klick:
	
	 1.Klick:   "info1"
	 2.Klick:   "info2"
	 3.Klick:   "info3"
	 4.Klick:   "info1"
	 etc.

	Verknüpfen Sie diese Methode per click EventHandler  
	mit dem Button Element "#b1".
