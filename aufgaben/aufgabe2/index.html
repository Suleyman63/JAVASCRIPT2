Demo 2 Arrays auslesen

 

	Um Arrays zu verstehen, muss man sie genau betrachten.
	1.
	Schreiben Sie eine Funktion die alle Namen 
	aus dem Array ausliest und ihn im p#info Element ausgibt.
	Die Ausgabe soll so aussehen: 
	
	#########################################################
	Fritz Fischer
	Hans Mayer
	Frieda Friedlich
	Hans Glück
	#########################################################
	
	Verwenden sie für den Zeilenumbruch das HTML Element <br>
	Verknüpfen Sie diese Funktion per click EventHandler 
	mit dem Button Element #b1.
	
	Leider sind die Arrays nicht wirklich gut strukturiert.
	Suchen Sie eine trotzdem eine systemische Lösung, die immer funktioniert,
	gleichgültig wieviele Namen Paare enthalten sind.
	
	Stöbern Sie in den Array Methoden nach einer teilweisen Lösung für dieses Problem.
	Eine gute Idee wäre z.B. Array.flat() einzusetzen.
	Versuchen Sie for Schleifen einzusetzen.	

	2.
	Schreiben Sie eine Funktion die den sichtbaren Inhalt 
	des p#info Element löscht.
	Verknüpfen Sie diese Funktion per click EventHandler  
	mit dem Button Element "#b2".
